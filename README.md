## 🔹 Project Description

Incorporated data analysis techniques with a focus on time series analysis to extract meaningful insights from sales data. 

Developed an interactive Power BI dashboard to visualize trends, track KPIs, and support accurate sales forecasting, helping in data-driven business decision-making.

## 🔹 Key Highlights
- Time series analysis for sales trends
- Sales forecasting using historical data
- Interactive dashboard with filters and KPIs
- Business insights for decision-making

## 🔹 Tools Used
- Power BI
- DAX
- Excel / CSV
